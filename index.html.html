How to read this table: Each column is one week of the year (Week 1 = first 7 days of the month). Irrig. Date = recommended next irrigation trigger date within that week based on the irrigation frequency (IF). Depth (mm) = gross depth to apply per irrigation event. Run time (hr) = how long to run sprinklers per set position (Ts). Shifts = number of lateral moves needed to complete the irrigation cycle. Weeks highlighted in amber are peak demand weeks — prioritise full application. Weeks in green have surplus rainfall — irrigation can be skipped or reduced.

Farmer Irrigation Advisory — SMS Composer

The SMS below is auto-generated from this session's calculations. It gives farmers the key scheduling decision: when to irrigate, how long to run the system, and the gross depth to apply. It is concise enough for a single SMS (≤160 chars per segment) and written in plain language.

What farmers need to know and why:

① Next irrigation date — so they mobilise labour and move laterals in time.
② Gross depth (mm) — tells them when to stop; avoids both under- and over-irrigation.
③ Set time Ts — how long sprinklers run per set position.
④ Number of shifts — so they know how many days/moves to complete the cycle.
⑤ Alert flag — warns if soil is near deficit or rainfall has reduced requirement.
